Para agilizar o atendimento

  1. Informe seu nome completo e o e-mail cadastrado no Skalli.
  2. Explique o que aconteceu e, se possível, envie prints da tela.
  3. Informe o modelo do celular, sistema operacional e versão do aplicativo, quando souber.
  4. Não envie dados de pacientes, prontuários, diagnósticos ou informações clínicas.
Atenção: o Skalli é uma ferramenta de gestão operacional de escalas. Não use os canais de suporte para enviar dados clínicos ou assistenciais de pacientes.
